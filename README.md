# Firewolf Map Manifests

This directory contains the pure-data JSON manifests for Firewolf maps, ingested by the client spawner and server validator.

## Maps
- **CTF-Slag**: Ski-friendly CTF map with opposing ridges and valleys.
- **CTF-Mesa**: Keep-hill CTF map with flat-topped mesas and a wide basin.
- **CTF-Glacier**: Vertical ice biome map with low-friction ski surfaces and a contestable central ice-tower.
